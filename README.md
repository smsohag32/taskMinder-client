### Task Minder

Task Minder is a web application that helps you manage and track your tasks effectively.

## Live Link

You can access the live version of Task Minder by following this link: [Task Minder Live](https://taskminder-sm.netlify.app/)

## Features

1. **Add Task**: Users can create and add new tasks to their task list.
2. **Edit Task**: Users can modify the details of existing tasks, such as the title, description, and status.
3. **Delete Task**: Users can delete unwanted tasks from their task list.
4. **Update Task Status**: Users can update the status of tasks to reflect their progress, such as "in progress," "completed," or "pending."
5. **Form Validation**: The application includes form validation to ensure that required fields are filled out correctly.
6. **Task Details**: The task details, including the title, status, and description, are displayed dynamically on the All Tasks page.

7. **Backend Technology**: The backend of the Task Minder website is built using Express.js, and NodeJs, providing a robust and scalable server-side solution.
8. **MongoDB**: Task Minder uses MongoDB for performing CRUD operations, allowing users to create, retrieve, update, and delete tasks through the provided APIs.
